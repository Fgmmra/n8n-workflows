## 📌 Sobre o projeto

Este repositório é um portfólio vivo de automações construídas com **n8n**, cobrindo processos reais de negócio em múltiplos domínios. Cada workflow foi desenvolvido com foco em resultado prático, boas práticas de arquitetura e reusabilidade.

> 🔄 **Este repositório é atualizado todos os dias.** Novos workflows são adicionados constantemente — volte sempre para conferir as novidades.

---

## 🗂️ Estrutura por Domínio

```
domains/
├── customer-service/       # Atendimento ao cliente e suporte
├── finance/                # Processos financeiros e relatórios
├── information-technology/ # TI, infraestrutura e DevOps
├── marketing/              # Campanhas, leads e automação de marketing
```

- **Webhooks** — gatilhos em tempo real

---

## 🚀 Como usar

1. Acesse a pasta do domínio desejado
2. Baixe o arquivo `.json` do workflow
3. No n8n, vá em **Import Workflow** e importe o arquivo
4. Configure as credenciais necessárias
5. Ative o workflow e pronto!
